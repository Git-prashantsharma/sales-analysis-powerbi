# Sales Analysis Dashboard 
This repo includes data analytics projects build on Power BI
Sales Analysis Dashboard in Power BI, designed to deliver executive-level insights across product performance, geographic trends, and segment profitability.
This report brings together multi-dimensional data storytelling with interactive visuals, enabling decision-makers to:
- Track sales growth and profit margins over time
- Compare performance across countries, segments, and products
- Identify top-selling SKUs and revenue drivers
- Pinpoint seasonal trends and MoM fluctuations

- Profit Margin: 11.25%
- Sales Growth (YoY): 18%
